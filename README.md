## 서비스 소개
### 하루 한 번 세상을 나누는 디지털 광장, 아고다

- 오늘의 논제 선택
- 논제에 대한 목소리 보기 / 남기기
- 상황에 따른 밸런스 리스트 보기
- 내가 참여한 밸런스 리스트 보기
- 논제 만들기
